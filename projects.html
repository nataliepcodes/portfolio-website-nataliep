RECENT PROJECTS

Below you can see information about some of the recent projects that I have developed. Some projects were part of completing front-end development certification and some developed out of personal interest.

Motivational Webpage

HTML ・ CSS ・ Vanilla Javascript
A famous motivational quote is interpreted as a recipe. This was my first coding project at the coding workshop. Starting coding by chance and coming from a non-technical background, I found it an amazing experience to be able to create something like that. This might be basic JavaScript but we all have to start somewhere! Frankly, I enjoyed developing this! This generated a lot of introspection about why exactly did I not get into this before. Since we cannot change the past. But can certainly create the future. Or at least learn something that we are interested in - I signed up for the next coding workshop level. The next project followed.

Weather app

HTML ・ CSS ・ Vanilla Javascript ・ API ・ AXIOS ・ Bootstrap ・ Font Awesome
Weather application developed as part of my final project at front-end development workshop. The Open Weather API is used to call a real-time data to display weather for cities all over the world. Geolocation API is used to display a current location of the user.

Hello Fall Webpage

HTML ・ CSS
Webpage with the effect of fixed photos. When the page is scrolled up and down only the text is moving, not the photos. Some call this a parallax effect, some disagree. I liked the effect when I saw it and enjoyed re-creating it. Best enjoyed on desktop and laptop devices.
If you are interested in the Parallax effect and looking for inspiration, feel free to check this article: iMPACT
